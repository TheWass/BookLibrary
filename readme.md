Book Library
--------
This app scans in ISBNs, goes off to an API to collect info about a book, and stores it in a sqlite database.

Dev Info
--------
`npm install -g expo-cli`
`npm install`
`expo start`

Improvements
-----
* Purge should clear the mem store
* Detect duplicates
* Database migration strategy

Features
------
* Edit book entry
* Sort
* Letter headers
* Publish year
